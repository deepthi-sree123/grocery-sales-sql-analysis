# Grocery Sales SQL Analysis

This project contains a small grocery sales database 
and SQL queries for data analysis including:

- SELECT, WHERE, ORDER BY
- GROUP BY & aggregate functions
- JOINS (INNER, LEFT)
- Subqueries
- Views
- Index creation

## Files in this repository

- database_setup.sql — database schema + 20-row sample data
- analysis_queries.sql — all analysis SQL queries
- screenshots/ — MySQL query result screenshots
